# Absolvment Hub

Roblox script hub for Absolvment game.

## Features

### Visuals
- Player ESP (Box, Name, Health, Distance, Tracers)
- NPC/Mob ESP (Box, Name, Health, Distance)

### Coming Soon
- Auto Farm
- Auto Combat
- Movement (Fly, Speed, Noclip)
- Teleports

## Structure

```
Absolvment/
├── Game/          # Backend modules
│   └── Visuals/   # ESP modules
├── GUI/           # UI library (Obsidian)
├── UI/            # Frontend tabs
└── main.lua       # Entry point
```

## Usage

Load the script in your executor:
```lua
loadfile("main.lua")()
```
